PengUIns CSS

Міні CSS бібліотека команди Penguins

Що це і для чого?

Якщо потрібно швидко сворити простий HTML документ, а на форматування та написання CSS немає часу - можна використати уже готові стилі форматування елементів

Як підключити?

Додати файл penguins-ui.css до свого проекту та підключити його в HTML файлі з допомогою тегу <link>:

<link rel="stylesheet" href="penguins-ui.css">

Як використовувати?

Потрібно додати клас .p_ui до тегу <body>. Після цього стилі автоматично будуть застосовані до елементів сторінки.

Примітки до форматування коду:

  • клас"code-block" додає властивість display: block
  • для виділення рядків використовується клас "code-string"
  • для виділення CSS класів використовується клас "code-class"
  • для створення відступів використовується клас "code-margin"
  • для виділення вмісту HTML тегу використовується клас "code-tag-inner"

Приклад HTML розмітки коду:

<code class="code-block">
<span class="code-class">.p_ui blockquote</span> {<br>
<div class="code-margin">font-size: <span class="code-string">18px</span>;<br>
margin: <span class="code-string">32px 16px 32px 48px</span>;<br>
padding: <span class="code-string">32px 8px 32px 16px</span>;<br>
font-family: <span class="code-string">'Inter', sans-serif</span>;<br>
font-weight: <span class="code-string">700</span>;<br>
</div><br>
}<br>
</code>

Результат:

.p_ui blockquote {
font-size: 18px;
margin: 32px 16px 32px 48px;
padding: 32px 8px 32px 16px;
font-family: 'Inter', sans-serif;
font-weight: 700;
}
